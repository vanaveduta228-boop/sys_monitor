# System Monitor

Легка утиліта на Python для моніторингу ресурсів системи в реальному часі.

## Можливості
- **Моніторинг:** CPU, RAM, використання диска, мережевий трафік, заряд батареї.
- **Гнучке налаштування:** Усі ліміти задаються через файл `config.json`.
- **Валідація:** Скрипт перевіряє наявність та коректність параметрів конфігурації перед запуском.
- **Система анти-спаму:** Сповіщення надсилаються лише при зміні стану (початок проблеми / нормалізація), що запобігає засміченню логів.
- **Безпечне завершення:** Коректна зупинка програми через `Ctrl+C`.

## Конфігурація:
Створіть або відредагуйте файл config.json у кореневій папці проекту перед запуском. Приклад структури:
{
    "cpu_limit": 80,
    "ram_limit": 85,
    "disk_min_gb": 10,
    "net_limit_gb": 100,
    "battery_min_percent": 20,
    "check_interval_seconds": 60
}

## Для запуску програми виконайте команду:
python main.py

Програма працюватиме в нескінченному циклі, контролюючи параметри системи. Для коректної зупинки натисніть Ctrl+C. Всі критичні повідомлення та логі зберігаються у файл monitor.log.
Проєкт створено для автоматизованого моніторингу та діагностики системних ресурсів.
