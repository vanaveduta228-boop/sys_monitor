# System Resource Monitor

Скрипт для безперервного моніторингу системних ресурсів (CPU, RAM, Диск, Мережа, Батарея) та перевірки доступності мережевих портів. Програма автоматично сповіщає користувача у разі перевищення заданих лімітів.

## Вимоги
* Python 3.x

## Встановлення та запуск

1. **Створіть віртуальне середовище:**
```bash
python -m venv venv
```

2. **Активуйте віртуальне середовище:**

*Для Windows:*
```bash
venv\Scripts\activate
```

*Для Linux/macOS:*
```bash
source venv/bin/activate
```

3. **Встановіть необхідні залежності:**
```bash
pip install -r requirements.txt
```

## Налаштування (config.json)
Перед запуском переконайтеся, що у кореневій папці проекту знаходиться файл `config.json`. У ньому задаються порогові значення для спрацьовування алертів. 

Приклад структури:
```json
{
  "cpu_limit": 85,
  "ram_limit": 80,
  "disk_min_gb": 10,
  "net_limit_gb": 100,
  "battery_min_percent": 20,
  "check_interval_seconds": 60,
  "server_to_check": "8.8.8.8:53"
}
```

## Запуск програми
Щоб запустити моніторинг, виконайте команду:
```bash
python main.py
```
Для коректного завершення роботи скрипта натисніть `Ctrl+C` у терміналі.
